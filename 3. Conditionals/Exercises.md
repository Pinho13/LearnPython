# Exercises

> Note: The symbol / is used to indicate a line separation, like a paragraph for each part of the output.

## 1. Even or Odd

Given an integer, print whether it is even or odd.

| Input | Output |
|:------|:-------|
| 8     | Even   |
| 15    | Odd    |

## 2. Positive, Negative or Zero

Given a number, print whether it is positive, negative, or zero.

| Input |  Output  |
|:------|:---------|
| 5     | Positive |
| -7    | Negative |
| 0     | Zero     |

## 3. Largest Number

Given two numbers, print the largest one.

| Input  | Output |
|:-------|:-------|
| 8 / 3  | 8      |
| -5 / 7 | 7      |

## 4. Leap Year

Given a year, determine whether it is a leap year.

> Note: A leap year is divisible by 4, except for years divisible by 100, unless they are also divisible by 400.

| Input |     Output     |
|:------|:---------------|
| 2024 | Leap year       |
| 1900 | Not a leap year |

## 5. ATM

Given:
- account balance
- withdrawal amount

Print whether the withdrawal can be completed.

|  Input   |         Output        |
|:---------|:----------------------|
| 100 / 30 | Withdrawal successful |
| 50 / 75  | Insufficient funds    |

## 6. Calculator

Given:
- two numbers
- an operator (`+`, `-`, `*`, `/`)

Perform the correct operation.

|    Input   | Output |
|:-----------|:-------|
| 5 / 3 / +  | 8      |
| 20 / 4 / / | 5.0    |

## 7. Number Ordering

Given three different numbers, print them in ascending order.

|    Input   |   Output   |
|:-----------|:-----------|
| 8 / 2 / 5  | 2 / 5 / 8  |
| -1 / 9 / 0 | -1 / 0 / 9 |

## 8. Rock, Paper, Scissors

Given two choices (`"rock"`, `"paper"`, or `"scissors"`), print the winner or if it's a tie.

|      Input      |     Output    |
|:----------------|:--------------|
| rock / scissors | Player 1 wins |
| paper / paper   | Tie           |

## 9. Triangle !?

Given the lengths of the three sides of a triangle, first determine whether they can form a valid triangle.

If they do, print whether the triangle is:

- Equilateral (all sides are equal)
- Isosceles (exactly two sides are equal)
- Scalene (all sides are different)

Otherwise, print `"Invalid triangle"`.

> Note: Three sides can form a valid triangle only if the sum of any two sides is greater than the remaining side.

|    Input   |      Output      |
|:-----------|:-----------------|
| 3 / 3 / 3  | Equilateral      |
| 2 / 3 / 10 | Invalid triangle |
